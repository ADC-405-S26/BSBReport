# A sample Trackman pitching dataset to highlight the funcitons in BSBReport

A sample Trackman pitching dataset to highlight the funcitons in
BSBReport

## Usage

``` r
trackman_data
```

## Format

### `trackman_data`

A data frame with 25 rows and 8 columns:

- PitcherId:

  Integer value denoting unique player id

- TaggedPitchType:

  Character denoting pitch type

- PitchCount:

  integer of pitch count

- InducedVertBreak:

  integer representing the IVB for each pitch

- HorzBreak:

  integer representing the HB for each pitch

- PlateLocHeight:

  integer of measured pitch height at front of home plate

- PlateLocSide:

  integer of measured pitch horizontal location at front of home plate

- RelSpeed:

  integer of measured pitch speed at release in mph
